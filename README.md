# Haztech Sprite Editor

*A modular Unity tool for creating and managing layered 2D character sprites.*

## About

Haztech Sprite Editor is a Unity editor extension designed to simplify the creation of modular 2D characters.

The tool allows developers to organise sprites into reusable layers, states, directions, and animations while generating data that can be used directly at runtime.

Originally created to support my own game projects, the editor is being developed as a reusable package for future projects.

## Planned Features

* Layer management
* Sprite state management
* Direction support (4-way, 8-way and custom)
* Animation editing
* Sprite preview
* Colour customisation
* Runtime integration
* Prefab generation
* Validation tools

## Technologies

* Unity
* C#
* UI Toolkit / IMGUI
* ScriptableObjects

## Status

🚧 Early Development

Core editor functionality is currently being implemented, with additional features planned throughout development.

## Goals

* Provide a flexible workflow for modular 2D characters.
* Separate editor tooling from runtime systems.
* Build a reusable package suitable for multiple Unity projects.
